# YouTube Data Pipeline & Dashboard

## Project Overview
This project collects trending video data from YouTube using API, processes it, and shows insights using a dashboard.

## Features
- Extract data from YouTube API
- Clean and transform data
- Store data
- Visualize using Streamlit

## Tech Stack
- Python
- Pandas
- Streamlit
- YouTube API

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
